##  Project Overview

The **Smart Event Dashboard** is a dynamic web application built using pure Vanilla JavaScript.  
It allows users to create, manage, and interact with event cards in real-time without using any external libraries or frameworks.

This project focuses on strengthening core concepts of:

- DOM Manipulation
- Event Handling
- Event Delegation
- Text & Style Manipulation
-Dynamic Element Creation


##  Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling (No frameworks used)  
- **Vanilla JavaScript** – DOM Manipulation & Events
